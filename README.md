A classic Snake game developed from scratch using Java. This project serves as a practical implementation of Object Oriented Programming (OOP) principles.
Key Features & Mechanics
Game Loop Architecture:Implemented a continuous game loop for real time rendering and frame updates
Collision Detection: Logic to handle snake to-wall and snake to body collisions efficiently.
Dynamic Data Structures: Handling of the snakes body segments as dynamically growing arrays.
Event Handling: Real time keyboard input listening for smooth directional control.
Developed to bridge the gap between theoretical data structures and applied software architecture focusing on clean code and logical flow
