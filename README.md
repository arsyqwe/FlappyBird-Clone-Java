A desktop implementation of the popular Flappy Bird game built entirely in Java. This project demonstrates applied 2D physics real time object rendering and state management within a custom game loop.
Simulated Physics: Mathematical implementation of gravity velocity and jump momentum (acceleration/deceleration) for realistic movement.
Precise Collision Detection: Axis-Aligned Bounding Box  intersection logic to detect bird to pipe and bird to ground collisions.
Dynamic Environment: Algorithmic generation of infinite obstacles  with randomized gap placements and automated memory cleanup.
State Management: Handled distinct game states smoothly (Start Screen, Active Gameplay, Game Over).
